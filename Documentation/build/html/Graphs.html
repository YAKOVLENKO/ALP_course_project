graphs.py¶

class Graph¶

Построение графа

Переменные:
  • gr_CommonFr – Граф, показывающий как много общих друзей между пользователем и его друзьями
  • gr_Group – Граф, показывающий как много друзей пользователя являются участниками определенного количества тех или иных его групп
  • gr_GroupSix – Граф, показывающий как много друзей пользователя являются участниками определенных групп
  • friends – Друзья определённого пользователя
  • last_add – Объекты, которые будут добавлены в граф gr_CommonFr при его изменении
  • pers_obj – Объект класса Profile
  • api – Сессия VK
  • token – Токен
