Profile.py
=============

.. py:class:: Profile
   
   Обработка полученных данных о пользователе и связь с текущим пользователем через интерфейс

   :var vk_data: Общие данные пользователя для обработки
   :var groups: Данные о группах пользователя для обработки
   :var id: Идентификационный номер пользователя
   :var fname: Имя пользователя
   :var lname: Фамилия пользователя
   :var mname: Отчество пользователя
   :var birth_day: День рождения пользователя
   :var birth_month: Месяц рождения пользователя
   :var birth_year: Год рождения пользователя
   :var city: Город, в котором проживает пользователь
   :var country: Страна, в которой проживает пользователь
   :var mobile_phone: Мобильный телефон пользователя
   :var skype: Связь с пользователем: Skype
   :var instagram: Связь с пользователем: Instagram
   :var twitter: Связь с пользователем: Twitter
   :var career: Места работы пользователя
   :var education: Места учёбы пользователя
   :var interests: Интересы пользователя
   :var relatives: Родственные связи пользователя
   :var likes: Посты, которые понравились пользователю
